# Лабораторна робота №2

## Мета
- Повторити та закріпити роботу з класами, інтерфейсами та успадкуванням у C#.
- Навчитись реалізовувати глибоке копіювання об'єктів та працювати з колекціями.

## Чого ми навчилися
- Створення та використання класів (`Person`, `Student`).
- Оголошення та реалізація інтерфейсів (наприклад, `IDateAndCopy`).
- Наслідування та поліморфізм: як розширювати базові класи і перевизначати методи.
- Робота з масивами/колекціями об'єктів (додавання, пошук, сортування).
- Практика з конструкторів, властивостей (`get`/`set`) і методів.

## Теоретичні відомості
- Клас — шаблон для створення об'єктів, що інкапсулює стан (поля) і поведінку (методи).
- Інтерфейс описує контракт без реалізації; класи, що реалізовують інтерфейс, зобов'язані надати реалізацію його методів.
- Успадкування дозволяє створити спеціалізований клас на основі загального, повторно використовуючи код.
- Глибоке копіювання створює повністю незалежну копію об'єкта разом із вкладеними об'єктами, на відміну від поверхневого копіювання.

## Пояснення реалізації (коротко)
- Файли проекту:
  - `Person.cs` — базовий клас з основними полями (ім'я, прізвище, дата народження).
  - `Student.cs` — наслідує `Person`, містить інформацію про оцінки/екзамени.
  - `Exam.cs` — структура/клас для збереження даних екзамену (назва, оцінка, дата).
  - `IDateAndCopy.cs` — інтерфейс, що описує методи для отримання дати та копіювання об'єкта.
  - `Program.cs` — точка входу: створює приклади об'єктів, демонструє додавання/копіювання і вивід результатів.

- Основні операції, реалізовані в коді:
  - Ініціалізація декількох студентів і їхніх екзаменів.
  - Демонстрація глибокого копіювання студента (щоб зміни в копії не впливали на оригінал).
  - Пошук і сортування студентів за різними критеріями (наприклад, середній бал).

## Як запустити
1. Відкрити папку проекту в терміналі.
2. Запустити команду:

```
dotnet run
```

## Приклад результату
- У консолі виводиться інформація про створених студентів, їхні екзамени, результати копіювання та приклади сортування/фільтрації.

Лабораторна робота №2 закріплює базові принципи ООП у C#, зокрема роботу з класами, інтерфейсами, наслідуванням і копіюванням об'єктів.
